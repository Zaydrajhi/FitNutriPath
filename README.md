# FitNutriPath – Intelligent Task & Appointment Management

An intelligent **task and appointment management module** developed for the **FitNutriPath healthcare and nutrition platform** as part of an academic project at ESPRIT (Ecole Supérieure Privée d'Ingénierie et de Technologies), Ariana, Tunisia.

## Features

* Task and appointment management for healthcare/nutrition professionals
* AI-powered prediction and automation using machine learning
* Interactive interfaces for data management and visualization
* Integration between Symfony backend and Java/JavaFX components
* Real-world healthcare workflow simulation

## Technologies

* Symfony (PHP)
* Java
* JavaFX
* php-ai / php-ml
* Machine Learning

## Project Period

March 2025 – May 2025

## Team & Methodology

Developed collaboratively using the **Scrum agile methodology**.
